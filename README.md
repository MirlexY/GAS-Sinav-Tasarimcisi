# 🎓 GAS Sınav Sistemi Tasarımcısı

GAS Sınav Sistemi Tasarımcısı, Google App Script (GAS) tabanlı sınav sonuç sorgulama sistemlerini saniyeler içinde tasarlamanıza ve yayınlamanıza olanak tanıyan bir masaüstü aracıdır.

## 🚀 Özellikler

- **Hızlı Kod Üretimi:** Google App Script (.gs) ve HTML (.html) dosyalarını tek tıkla hazırlar.
- **Yüksek Kaliteli Arayüz:** Modern, responsive ve koyu tema (Dark Mode) destekli sorgulama ekranları üretir.
- **Excel/CSV Entegrasyonu:** Veri başlıklarını otomatik çeker, manuel hatayı sıfıra indirir.
- **Görsel Kılavuz:** Uygulama içinde yüksek çözünürlüklü, adım adım kullanım rehberi içerir.
- **Otomatik Güncelleme:** GitHub üzerinden sürüm kontrolü yaparak her zaman en güncel sürümü kullanmanızı sağlar.
- **Setup Kurulumu:** Windows uyumlu profesyonel yükleyici desteği.

## 🛠️ Kurulum ve Kullanım

1. Uygulamayı çalıştırın.
2. Excel veya CSV dosyanızı sürükleyip bırakın.
3. Sınav başlığı, sayfa adı ve sütun eşleştirmelerini yapın.
4. "🚀 Kodları Üret" butonuna basın.
5. Oluşan kodları Google Apps Script projenize kopyalayın ve yayınlayın.
